# Code of Conduct

This project is a small, open, good-faith effort. The rules are short.

**Be decent.** Be respectful and constructive. Assume good intent. Critique ideas and code,
not people. No harassment, personal attacks, discrimination, or bad-faith behavior of any kind.

**Stay on mission.** This is a transparency project about apartment data. Keep contributions
and discussion aimed at that.

**Scrape responsibly.** Contributions target landlord-direct, public availability feeds only.
Anything that requires an account, defeats an access control, or is designed to overload a
site is out of scope and will not be merged.

Maintainers may edit, remove, or reject any contribution, issue, or comment that violates this,
and may block repeat offenders.

Report a problem to Paul Jump on X at [@paulljump](https://x.com/paulljump).
